# Trip App (Social Travel Companion)

Trip App is an Android application designed for travel enthusiasts to discover new destinations, rate and review locations, and connect with a community of fellow travelers.

## Features

- **Discover Destinations:** Explore a curated gallery of travel destinations with detailed descriptions and beautiful imagery.
- **User Experiences:** Read and write ratings or reviews to gain and share authentic insights into various locations.
- **User Authentication:** Secure Sign Up and Login functionality for personalized experiences.
- **Search:** Easily search for specific places or types of travel experiences.
- **Social Integration:** Connect with the community through integrated social media links (Instagram, Twitter, etc.).
- **User Profiles:** Manage your personal traveler profile, ratings, and contributions.

## Screenshots / UI Components

The application features a modern interface with both Jetpack Compose and traditional XML-based layouts, including:
- **Main Dashboard:** A central hub for discovering "Places you might like" and "The best ones".
- **Login/Sign-up Screens:** Clean and intuitive authentication forms.
- **Trip Details:** Dedicated views for specific travel experiences, including ratings and reviews.

## Technologies Used

- **Language:** Kotlin
- **UI Frameworks:** - [Jetpack Compose](https://developer.android.com/jetpack/compose) (Modern declarative UI)
    - [Android XML Layouts](https://developer.android.com/guide/topics/ui/declaring-layout) (Traditional View system)
- **Architecture:** Material Design 3
- **Build System:** Gradle (Kotlin DSL)
- **Libraries:**
    - AndroidX Core & AppCompat
    - ConstraintLayout
    - Lifecycle Runtime
    - Material 3

## Project Structure

- `app/src/main/java`: Contains the Kotlin source code, including `MainActivity.kt` and UI themes.
- `app/src/main/res/layout`: Contains XML layout files for various screens (Login, Sign-up, Main, Profile, etc.).
- `app/src/main/res/drawable`: Contains image assets and drawable resources.
